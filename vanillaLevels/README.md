# Vanilla Level Set

This Level Set contains the regions for the base game.
This repository may be updated from time to time with work-in-progress levels.

Feel free to build these locally if you'd like to help playtest them! Any feedback is appreciated, so the levels can be the best they can be. (though note newer regions may utilise objects or mechanics that are not present in the release build. you may need to switch over to a testing branch)
